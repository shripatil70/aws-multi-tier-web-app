# 🚀 Multi-Tier Web Application on AWS

## 📌 Overview
This project demonstrates a **2-tier cloud web application** deployed on AWS using EC2 instances.  
It includes a frontend hosted on Nginx and a backend REST API built with Flask.

---

## 🏗️ Architecture

Frontend (Nginx on EC2)  
        ↓ HTTP Request  
Backend (Flask API on EC2)

---

## ⚙️ Tech Stack

- AWS EC2 (Frontend & Backend servers)
- Nginx (Web server)
- Flask (Python backend API)
- HTML, CSS, JavaScript
- Linux (Amazon Linux 2023)

---

## 🚀 Features

- Fully deployed on AWS cloud
- Frontend hosted using Nginx
- Backend REST API using Flask
- Secure communication using AWS Security Groups
- Real-time API integration between frontend and backend
